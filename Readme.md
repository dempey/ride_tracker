# Spring JDBC Sample Project

When using this project:

 * The unit tests were used as part of the learning/development proecess so you should run mvn builds with tests turned off (mvn clean install -DskipTests=true).
 * This project is the result of running the Pluralsight course "Building Applications Using Spring JDBC", by Bryan Hansen
 * See create_ride_table.sql file for script that creates the "ride" table.
